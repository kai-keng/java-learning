# Spring Security

Spring Security是一个功能强大且高度可定制的身份验证和访问控制框架。 它是用于保护基于Spring的应用程序的实际标准。

该篇章会一步步讲解如何使用 Spring Security 完成一个功能完整的登录权限模块，让你在看完以后能够跟着快速的搭建出登录模块。
